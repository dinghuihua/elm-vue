  * html、css3、
  * Javascript、es6
  * stylus
  * webpack
  * better-scroll

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```

## Start a server
```
# build
npm run build

# start-up
node prod.server.js

# Listening in at http://localhost:9001
```
